## WAN Security Configuration

The following adjustment was required due to the lab being behind a home router:

| Setting | Value |
|---|---|
Block RFC1918 Private Networks | Disabled |
Block Bogon Networks | Enabled |

Reason:

The pfSense WAN interface receives a private IP address from the home router (192.168.x.x).

Blocking RFC1918 networks would prevent WAN communication in this environment.
